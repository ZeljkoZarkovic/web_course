Šta je WEB?

U ovom kursu naučićete o:

Internet i WWW

Internet je globalna mreža koja povezuje digitalne uređaje, korisnike iz različitih domena. World Wide Web omogućava dobijanje različitih datoteka - Web stranice, audio fajlova, kompjuterskih programa sa Interneta putem Hipertext Transfer Protocol (HTP).

HTML5 i CSS3

HTML (HyperText Markup Language)je osnovni jezik za kreiranje strukture web stranica, dok CSS (Cascading Style Sheets) definiše sam izgled i stil elemenata na tim stranicama. Zajedno omogućavaju kreiranje preglednih, estetskih i funkcionalnih web sajtova.

Java Script

JavaScript je jezik na strani klijenta (client-side scripting language) koji omogućuje kreiranje interaktivnih web stranica i sajtova.Izvršava se na web pretraživaču i omogućuje dinamičko menjanje sadržaja bez potrebe za ponovnim učitavanjem stranice.

XML

XML - Extensible Markup Language je jezik za razmenu i skladištenje podataka. Omogućava strukturisano predstavljanje informacija koje mogu koristiti različite aplikacije i sistemi.
