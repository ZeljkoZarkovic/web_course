JavaScript

Šta je to JavaScript?

JavaScript je skript jezik na strani klijenta (client-side scripting language) koji omogućava autorima Web stranica da razvijaju interaktivne Web stranice i sajtove. Skriptovanje na strani klijenta (client-side scripting) se odnosi na programski jezik koji se izvršava u okviru web pretraživača (na klijent sloju) umesto na Web serveru (na aplikativnom obrade). Deo pretraživača koji izvršava kod skript jezika naziva se skript mašina (scripting engine) pretraživača. Skript mašina je samo jedna vrsta prevodioca, gde se pojam prevodilac generalno odnosi na bilo koji program koji izvršava kod skript jezika. Kada mašina za skritovanje učita Web stranicu, ona prevodi bilo koji program pisan u skript jezicima, kao što je JavaScript. Web pretraživač koji sadrži skript mašinu zove se skript domaćin (scripting host).

    JavaScript omogućava Web programerima da obavljaju zadatke kao što su:
  • Čitanje tagova (elemenata) iz dokumenata i pisanje novih elemenata i teksta u dokumentima
  • Manipulisanje ili premeštanje teksta
  • Obavlljnje matematičkih proračuna nad podacima
  • Reagovanje na događaje, kao što je npr. korisnikov klik na dugme
  • Primanje tekućih podataka i vremena od korisnikovog računara ili kada je poslednji put dokument bio modifikovan
  • Određivanje veličine ekrana korisnika, verzije pretraživača, ili rezolucije ekrana
  • Preduzimanje radnje u skladu sa uslovima (npr. validacija), kao što su upozoravanje korisnika ako unese pogrešnu informaciju u formu

JavaScript alati: Adobe Dreamweaver, MS Expression Web, NetBeans, Xcode, MS Visual Studio, Aptana, vi and emacs, Komodo, Notepad++ itd.
JavaScript može biti ugrađen u stranicu ili stavljen u eksternu skript datoteku (slično kao kod CSS-a). Skriptovi se dodaju na web stranicu unutar <script> elementa. Tip atributa na početku taga <script> ukazuje koji skript programski jezik se nalazi unutar elementa, tako da za JavaScript koristite vrednost text / javascript.

    Postoje tri mesta (načina) gde možete staviti vaše Java skriptove, a jedan XHTML (HTML) dokument može da koristi sva tri, jer ne postoji ograničenje na broj skripti koje jedan dokument može da sadrži:
  • U okviru <head> tag-a: Ovi skriptovi ćе biti pozvani kada ih aktivira događaj.
  • U <body> tag-u: Ove skriptovi ćе raditi kada se stranica učita.
  • U eksternoj datoteci.

Takođe možete da smestite JavaScript kod u eksternom dokumentu koji ima ekstenziju .js. Kada postavite JavaScript u eksternu datoteku, potrebno je da koristite atribut src u <script> elementu.
Primer: <script type="JavaScript" src="scripts/validation.js"></script>
Document Object Model (DOM) je interfejs koji omogućava pristup i manipulaciju HTML dokumentima pomoću JavaScript-a. Predstavlja web stranicu kao strukturu objekata — svaki element (tekst, slika, forma, link...) postaje objekat sa svojim svojstvima, metodama i događajima.

    Kroz DOM možemo:
  • Čitati i menjati sadržaj i stilove (npr. promena boje pozadine),
  • Pozivati metode koje vrše akcije na stranici (npr. ispisivanje teksta),
  • Reagovati na događaje (npr. klik na dugme).

DOM je standard koji podržavaju svi moderni pregledači i ključan je za dinamičku manipulaciju web stranicama.

Svojstva Document objekta: alinkColor, bgcolor, fgcolor, lastModified, linkColor, referrer, title, vlinkColor.
Metodi Document objekta: write(string), writeln(string).
Svojstva i metodi FORM objekata: action, length, method, name, target; reset() i submit().
Svojstva FORM elemenata: checked, disabled, form, length, name, selectedIndex, type, value.
Metodi Form elemenata: blur(), click(), focus(), select().
Osnove JavaScript-a
Promenljiva(variable)se koristi za skladištenje nekih informacija. Predstvalja delić memorije računara gde možete da memorišete brojeve, stringove (koji su niz znakova), ili reference na objekte. U promenljive spadaju: Integer (celi brojevi), Floating point number (brojevi sa pokretnim zarezom), Boolean (tačno/netačno), Text (odnosi se na string podatke), Nizovi i objekti.
Operatori (operators) izvršavaju funkcije nad promenljivim. Postoje različite vrste operatora: Aritmetički operatori, Operatoori dodeljivanja, Operatori poređenja, Logički operatori, String operatori.
Funkcije (functions) su delovi skripta koji su grupisani zajedno u cilju realizacije određenog zadatka.
Uslovne naredbe (conditional statements) omogućavaju da izvršite različite akcije koje se baziraju na uslovu. Vrste uslovnih naredbi: if naredbe (koriste se kada želite da se skript izvrši ako je uslov tačan), if...else (koriste se kada želite da izvršite jedan set koda ukoliko je uslov tačan, a drugi ako je netačan), switch (koriste se kada želite da izaberete jedan blok koda od mnogih u zavisnosti od situacije).
Petlje (loops) se mogu podesiti tako da se blok koda izvršava određeni broj puta, ili dok je uslov zadovoljen.
Validacija forme - Postoji nekoliko načina za validaciju formulara. Kada korisnik klikne na dugme za slanje, aktivira se onsubmit događaj na <form> elementu, koji poziva funkciju za validaciju - smeštenu u skripti ili zaglavlju dokumenta. Ta funkcija treba da vrati true kako bi se podaci poslali. Ako postoji greška, vraća false i forma se ne šalje, već prikazuje korisniku gde je problem u unetim podacima.

Šta je to jQuery?

jQuery je jedna od najpopularnijih JavaScript biblioteka u upotrebi danas koju koristi 65% od najposećenijih 10 miliona sajtova na vebu. jQuery uključuje sledeće funkcije: Događaje, efekte i animacije, Ajax, JSON parsiranje, mašinu za selektovanje, DOM manipulacija zasnovana na CSS-u, objekte za asinhronu obradu itd. Karakteristike jQuery biblioteke: moćna biblioteka, mala veličina datoteka, odlična podrška za animacije, lakši rad sa AJAX-om, podrška za različite web brauzere, lakši rad sa CSS-om, proširivost, besplatna i otvorenog koda.
Pozivanje jQuery biblioteke
JQuery biblioteka se može preuzeti i postaviti na sopstveni web sajt ili referencirati sa nekog servera za opsluživanje sadržaja (CDN - Content Delivery Network)
<script src="http://code.jquery.com/jquery-latest.js"></script>
<script type="text/javascript" src="jquery-1.10.2.min.js"></script>
jQuery UI Toolkit - Ovaj okvir sadrži neke interesantne funkcionalnosti koje uključuju nove vizuelne elemente (widgets), skup ikonica, kao i mehanizam za lako generisanje atraktivnih CSS stilova.

    Preuzimanje i konfigurisanje biblioteke:
  • Odabrati (ili kreirati) temu
  • Preuzeti temu
  • Odabrati elemente
  • Preuzeti datoteku (zip)
  • Instalirati sadržaj zip datoteke u radni direktorijum
  • Ako instalirate više tema, kopirajte samo podatke za dodatne teme
  • Povežite se sa CSS datotekama
  • Povežite se sa JS datotekama

jQuery projekti
jQuery UI predstavlja pripremljeni skup interakcija korisničkog interfejsa, efekata, vidžeta i tema koji su kreirani nad jQuery bibliotekom.
jQuery Mobile je HTML5-zasnovani korisnički interfejs dizajniran za izgradnju prilagodljivih web sajtova i aplikacija za mobilne telefone, tablete i desktop uređaje.
QUnit je moćan i lak za korišćenje JavaScript okvir za testiranje jedinica (unit-a). Može se koristiti za testiranje bilo kakvog JS koda.
Sizzle je JavaScript CSS selektor mašina. Podržava CSS 3 selektore, kao i mnoge dodatne selektore. Podržava i Unicode.

JavaScript okviri

JavaScript okvir predstavlja okvir za web aplikacije pisane u JavaScript-u. JS okvir opisuje strukturu aplikacije i pruža programerima način da organizuju kod i učine aplikaciju fleksibilnijom i skalabilnijom. Najpoznatiji JS okviri su: Angular, React, Ember, Meteor, Backbone, Knockout, Vue, D3, Aurelia itd.
Angular je okvir za razvoj web aplikacija otvorenog koda i zasnovan je na TypeScript jeziku. Njime upravlja Google tim, kao i velika zajednica programera i kompanija. Podržava kreiranje progresivnih web aplikacija, izbornih mobilnih aplikacija, kao i aplikacija za desktop operativne sisteme. Nudi dobru brzinu i performanse kroz templejt rendering i razdvajanje koda (učitava se samo kod potreban za renderovanje učitanog pogleda). Poseduje i podršku za atomatizaciju testiranja, kao i za pristupačnost (accessability). Angular implementira MVC arhitekturu radi razdvajanja komponenti za prezentaciju, podatke i logičkih komponenti.

Bootstrap

Bootstrap predstavlja besplatan skup alata za kreiranje web sajtova i web aplikacija. Baziran je na HTML i CSS šablonima za tipografiju, kreiranje formulara, dugmadi, navigacionih i ostalih komponenti, kao i razne šablone i proširenja za JavaScript. Cilj Bootstrap framework-a je olakšavanje programiranja za web. Takodje, Bootstrap je framework za web aplikacije, tj. softverski framework koji je dizajniran tako da podrži razvoj dinamičkih web sajtova i web aplikacija. Bootstrap je kompatibilan sa najnovijim verzijama web pregledača.
Bootstrap mrežni sistem koristi niz kontejnera, vrsta i kolona za aranžiranje i poravnanja sadržaja na stranici. Izgrađen je pomoću Flexbox-a i potpuno je prilagodljiv za različite uređaje.

    Bootstrap podržava tri vrste kontejnera:
  • .container - setuje max-width na svakoj prelomnoj tački
  • .container-fluid - to je 100% širine na svim prelomnim tačkama
  • .container-{breakpoint} - to je širina 100% sve do određene prelomne tačke
